## GELU / SILU / GELU_ERF / GELU_QUICK / SILU_BACK

**Not in `unary-ops.cpp`.** Implemented under **`ops.cpp`**: row-parallel loops call **`vec.h`** helpers:

- **GELU (tanh form):** **`ggml_vec_gelu_f32`** / **`ggml_vec_gelu_f16`** -- f32: `0.5f*x*(1+tanh(SQRT_2_OVER_PI*x*(1+GELU_COEF_A*x*x)))` with **`GELU_COEF_A=0.044715f`**, or **fp16 table** path when **`GGML_GELU_FP16`**; f16: **128K table** `ggml_table_gelu_f16` indexed by input half bits.
- **GELU_ERF:** **`ggml_vec_gelu_erf_f32`** -- `0.5f*xi*(1+erff(xi*SQRT_2_INV))`.
- **GELU_QUICK:** **`ggml_gelu_quick_f32`** = `x/(1+exp(GELU_QUICK_COEF*x))`, **`GELU_QUICK_COEF=-1.702f`**; optional fp16 table when **`GGML_GELU_QUICK_FP16`**.
- **SILU:** dispatcher calls **`ggml_vec_silu_f32`** (**`vec.cpp`**: AVX2 **ggml_v_silu** on wide chunks + scalar tail) / **`ggml_vec_silu_f16`** in **`vec.h`** as `x/(1+exp(-x))` in float.
- **SILU_BACK:** **`ggml_vec_silu_backward_*`** in **`vec.h`** / **`vec.cpp`** (see **`ops.cpp`** `ggml_compute_forward_silu_back_*`).
