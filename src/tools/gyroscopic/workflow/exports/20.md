# Export 20: `external/llama.cpp/ggml/src/ggml-gyroscopic/common.h` and `ggml-cpu-impl.h`

## `common.h` (gyroscopic)

- **Includes:** `ggml.h`, `traits.h`, **`ggml-cpu-impl.h`**, `ggml-impl.h`, **`simd-mappings.h`** (SIMD width macros consumed by **`vec.h`**, **`simd-gemm.h`**, arch quants).
- **Flash-attn tile constants:** **`GGML_FA_TILE_Q`**, **`GGML_FA_TILE_KV`** both **64**; exposed via **`ggml_fa_tile_config`** (C++ only).
- **`type_conversion_table<T>`** -- `to_f32` / `from_f32` for **`float`**, **`ggml_fp16_t`**, **`ggml_bf16_t`**, **`int32_t`**; used by **`binary-ops.cpp`** / **`unary-ops.cpp`**.
- **`get_thread_range(params, src0)`** -- splits **`ggml_nrows`** across **`ith`/`nth`** (shared row-parallel idiom).
- **Small F32/F16/BF16 helpers:** `f32_to_f16`, `f16_to_f32`, `f32_to_bf16`, `bf16_to_f32`, etc.

**`block_q8_0`:** **not** defined here; lives in **`ggml/src/ggml-common.h`** (`exports/6.md`).

## `ggml-cpu-impl.h` (CPU internal)

- **`struct ggml_compute_params`** -- `ith`, `nth`, `wsize`, `wdata`, `threadpool`, **`use_ref`**.
- **Compiler / ISA glue:** MSVC **`__FMA__` / `__F16C__`** when AVX2/AVX512; SSE3/SSSE3 under MSVC+AVX; **`m512bh` / `m512i`** casts.
- **ARM NEON:** large block of **polyfills** for 32-bit ARM, **`ggml_vld1q_*`**, **`ggml_vqtbl1q_*`**, etc. (quant/repack helpers consume these on ARM).
- **SVE / s390 / other:** conditional includes and feature macros at top of file.

**SIMD helper macros** for x86 AVX2 live primarily in **`simd-mappings.h`** (included from **`common.h`**), not as a duplicate block inside **`ggml-cpu-impl.h`**.

## Gyroscopic action

- **codec.c:** Use **`GGML_FA_TILE_*`** only if matching flash tile shapes; quants use **`simd-mappings.h`** + **`ggml-cpu-impl.h`** NEON helpers only on ARM builds.
- **log.md:** **`common.h`** = C++ glue + FA tile sizes + type conversion templates; **`ggml-cpu-impl.h`** = **`ggml_compute_params`** + portability intrinsics.
