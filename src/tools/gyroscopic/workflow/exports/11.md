# Export 11: `external/llama.cpp/ggml/src/ggml-gyroscopic/vec.h`

## `vec_dot` declarations (float family only)

Non-inline symbols implemented in **`vec.cpp`**:

- `ggml_vec_dot_f32` -- gyro hook site (`exports/10.md`).
- `ggml_vec_dot_bf16`
- `ggml_vec_dot_f16`

**Quantized** `ggml_vec_dot_*` pairs (e.g. `ggml_vec_dot_q8_0_q8_0`) are declared in **`quants.h`** / implemented in **`quants.c`** + arch TUs, not in this header.

## Typedef for `vec_dot` pointers

**Not in `vec.h`.** Per-type `vec_dot` function pointers live on **`ggml_type_traits_cpu`** in **`ggml-cpu.c`** (wired to the concrete symbols above or to `quants` entry points).

## Notable inline dots / unrolls

- **`ggml_vec_dot_f16_unroll`** -- `GGML_VEC_DOT_UNROLL` (default **2**) parallel F16 row dots vs one shared `y`; large SIMD block under `GGML_SIMD` (SVE / AVX2 / etc. in header).
- **`ggml_vec_norm_f32`** -- `ggml_vec_dot_f32(n, s, 0, x, 0, x, 0, 1)` then `sqrtf(*s)` (inherits F32 dot hook when `x` is the only operand pair).

## Other inline groups (one-line taxonomy)

- **Set/cpy:** `ggml_vec_set_*`, `ggml_vec_cpy_*` for i8/i16/i32/f16/bf16/f32.
- **Elementwise f32/f16:** `add`, `sub`, `mul`, `div`, `acc`, `neg`, `add1`, `acc1`.
- **Fused:** `ggml_vec_mad_f32`, `ggml_vec_mad_f16`, `ggml_vec_mad_f32_unroll`, `ggml_vec_mad1_f32` (`GGML_VEC_MAD_UNROLL` **32** for unroll variant).
- **Reduce:** `ggml_vec_sum_f32`, `ggml_vec_sum_*_ggf`, `ggml_vec_max_f32` (SIMD inline bodies in this header). **`ggml_vec_soft_max_f32` / `ggml_vec_log_soft_max_f32`** are **declared** here; **definitions** live in **`vec.cpp`**.

## Macros

- `GGML_VEC_DOT_UNROLL`, `GGML_VEC_MAD_UNROLL`, `GGML_SOFT_MAX_UNROLL`, `GGML_GELU_*` -- tune unrolling / tables shared with other TUs.

## Gyroscopic action

- **Hooks:** Only **`ggml_vec_dot_f32`** is bridged in **`vec.cpp`**; header inlines (`add`, `mad`, `max`, `sum`, `soft_max` inlines) are **stock** unless separately instrumented.
- **log.md:** Quantized dots = **`quants.h`** + arch; this header = **F32/F16/BF16 vector utilities** feeding matmul and elementwise ops.
