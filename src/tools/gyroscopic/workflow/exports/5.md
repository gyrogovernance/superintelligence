## Stock AVX2 inner sequence (`__AVX2__`, ~1052-1069)

- `nb = n / QK8_0` ( **`QK8_0` = 32** ints per block ).
- `acc = _mm256_setzero_ps()`.
- Per block `ib`:
  - `d = _mm256_set1_ps( GGML_CPU_FP16_TO_FP32(x[ib].d) * GGML_CPU_FP16_TO_FP32(y[ib].d) )`  (**fp16 scales -> float**, combined per block).
  - `qx = _mm256_loadu_si256(&x[ib].qs)`  (32 x int8)
  - `qy = _mm256_loadu_si256(&y[ib].qs)`
  - `q = mul_sum_i8_pairs_float(qx, qy)`  -> **8 float lanes** (see below).
  - `acc = _mm256_fmadd_ps(d, q, acc)`
- `sumf = hsum_float_8(acc)`; optional scalar tail loop for leftover blocks if any; `*s = sumf`.

### `mul_sum_i8_pairs_float` (AVX2, ~134-145)

- If `__AVXVNNIINT8__`: `_mm256_dpbssd_epi32` -> `cvtepi32_ps`.
- Else: `_mm256_sign_epi8(x,x)` -> `ax`; `_mm256_sign_epi8(y,x)` -> `sy`; then **`mul_sum_us8_pairs_float(ax, sy)`**.

### `mul_sum_us8_pairs_float` (AVX2, ~117-130)

- If AVX512VNNI+VL: `_mm256_dpbusd_epi32`.
- Elif `__AVXVNNI__`: `_mm256_dpbusd_avx_epi32`.
- Else (baseline AVX2): **`_mm256_maddubs_epi16(ax, sy)`** then **`sum_i16_pairs_float(dot)`** (`_mm256_madd_epi16(ones, x)` + `cvtepi32_ps`).

**Instruction grammar (baseline AVX2):** signed int8 via sign-mask -> **256-bit maddubs** -> int16 dot lanes -> **madd_epi16** reduce -> float 8-wide.

## 32 vs 64 grain

- **Natural block:** **32** int8 values per `block_q8_0` (`QK8_0`).
- Outer loop is **one block per iteration** (`ib++`); no 64-wide fused pair in this function (two consecutive Q8_0 blocks could be fused in a future kernel; stock is 32-wide loads).
