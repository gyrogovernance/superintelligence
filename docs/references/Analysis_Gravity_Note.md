## Appendix D: Derivation of the Exact Vacuum Solution and Self-Energy

The main text claims the exact point-mass exterior solution closes analytically and yields a finite self-energy. This appendix provides the derivations.

**D.1 The Exact Solution**

For a point mass, the potential satisfies the ordinary differential equation:

```text
dψ/ds = −exp(g₁ψ)/s²
```

where ψ is the gravitational depth and s is the areal radius. Separating variables and integrating yields:

```text

(−1/g₁) exp(−g₁ψ) = 1/s + C
```

Applying the boundary condition ψ → 0 as s → ∞ forces C = 0. Solving for ψ gives the exact closed form:

```text

```

This reduces to ψ = 1/s in the Newtonian limit g₁ → 0. The solution remains real and finite for all s > 1/g₁.
